# Netflix Churn Prediction Project

## Overview
This project analyzes user behavior and predicts churn using machine learning.

## Features
- Exploratory Data Analysis (EDA)
- Churn Prediction Model (Random Forest)
- SMOTE for class imbalance
- Feature Importance Analysis
- Streamlit Dashboard

## Results
- Accuracy: 83%
- Churn Recall: 74%
- Key factors: pricing, ratings, engagement

## Tools
Python, Pandas, Scikit-learn, Streamlit
